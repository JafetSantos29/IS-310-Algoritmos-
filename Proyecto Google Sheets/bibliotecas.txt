Comando: pip install google-auth google-auth-oauthlib google-auth-httplib2 google-api-python-client
Para instalar bibliotecas:
google-auth				herramientas para autenticación y autorización para las APIs de Google
google-auth-oauthlib		herramientas adicionales para trabajar con el protocolo OAuth2 de forma más fácil en Python.
google-auth-httplib2		proporciona herramientas para manejar la autenticación y autorización de solicitudes HTTP utilizando Google OAuth2.
google-api-python-client	Esta librería proporciona una forma fácil de interactuar con las APIs de Google utilizando Python
